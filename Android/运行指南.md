# 智舞流光 (DanceFlow) 运行指南

## 项目概述

智舞流光是一款舞蹈学习辅助 Android 应用，包含社区交流、学习资源、动作分析等功能。

### 技术栈

**前端 (Android)**
- Kotlin
- Jetpack Compose
- Retrofit2 (网络请求)
- Coil (图片加载)
- CameraX (相机)
- ExoPlayer (视频播放)

**后端 (Spring Boot)**
- Java 17
- Spring Boot 3.2.0
- Spring Security + JWT
- MySQL 8.0
- JPA/Hibernate

## 数据库配置

### 1. 安装 MySQL

确保已安装 MySQL 8.0 或更高版本。

### 2. 创建数据库

```bash
# 启动 MySQL 服务
# Windows:
net start mysql

# 登录 MySQL
mysql -u root -p
```

### 3. 执行初始化脚本

```bash

source C:/Users/ruanyuantao/Desktop/智舞流光/软件工程/DanceFlow-Backend/src/main/resources/sql/init_database.sql
```

或者直接运行：

```bash
mysql -u root -p < C:/Users/ruanyuantao/Desktop/智舞流光/软件工程/DanceFlow-Backend/src/main/resources/sql/init_database.sql
```

### 4. 修改配置文件

编辑 `application.yml` 修改数据库连接信息：

```yaml
spring:
  datasource:
    url: jdbc:mysql://localhost:3306/danceflow?useSSL=false&allowPublicKeyRetrieval=true&serverTimezone=UTC
    username: root
    password: 你的密码
```

## 常见问题

### 后端启动失败

**问题：** 端口 8080 被占用

**解决方案：**
```bash
# Windows 查找占用进程
netstat -ano | findstr :8080
# 结束进程
taskkill /PID 进程号 /F
```

或修改 `application.yml` 中的端口号。

### 前端无法连接后端

**问题：** 模拟器无法访问本地服务器

**解决方案：**
- 确认后端正在运行
- 使用 `10.0.2.2` 代替 `localhost`（模拟器专用）
- 如果使用真机，确保手机和电脑在同一网络，使用电脑的局域网 IP

### 数据库连接失败

**问题：** Unable to connect to database

**解决方案：**
1. 确认 MySQL 服务正在运行
2. 检查用户名密码是否正确
3. 确认数据库 `danceflow` 已创建

## 开发说明

### 动作分析模型集成

当前项目的动作分析功能预留了接口，位于：

- 后端：`AnalysisService.java` 的 `analyzeVideo()` 方法
- 前端：`AnalysisResultScreen.kt`

集成步骤：

1. 准备好模型分析服务（Python Flask/FastAPI）
2. 在后端 `AnalysisService` 中调用模型服务
3. 将分析结果保存到数据库
4. 前端获取并展示分析结果
